Projetos

Aprendizado por Projeto Integrado (API) Fatec - Turma DSM 2024 | 2

O projeto teve como objetivo o desenvolvimento de uma plataforma web para o acompanhamento do desempenho dos vereadores em exercício na cidade de São José dos Campos. A plataforma permite visualizações gráficas comparativas e possibilita que os cidadãos interajam, deixando comentários e feedbacks sobre o desempenho dos representantes públicos.

Ferramentas Utilizadas

Minhas Contribuições como Product Owner

  • Proposta de Modelo de Projeto: Como Product Owner, propus um modelo de projeto que priorizava a experiência do usuário e a simplicidade de navegação, aplicando uma alternativa eficaz e suficiente em relação a abordagem dos outros grupos de API da classe. Essa proposta facilitou o trabalho de raspagem de dados da equipe, eliminando informações irrelevantes e garantindo que apenas dados essenciais fossem coletados, o que resultou em uma aplicação mais leve e eficiente.
  • Desenvolvimento e Estruturação do Backend: Implementei e organizei toda a infraestrutura de backend utilizando Flask, desde a criação do arquivo principal app.py até a definição das rotas e filtros para todas as telas da aplicação. Estruturei o modelo HTML inicial com Flask, que serviu como padrão para os demais desenvolvedores, estabelecendo uma base uniforme e organizada. Além disso, realizei um estudo aprofundado sobre a manipulação de arquivos JSON e compartilhei esse conhecimento com o grupo, proporcionando uma base sólida para que todos pudessem trabalhar de maneira eficiente com esse formato de dados.
  • Design e Desenvolvimento das Páginas Principais: Fui responsável pelo design e implementação de diversas páginas essenciais da plataforma. Propus e desenvolvi a página "Proposições Aprovadas", ajustando o conteúdo e layout dessa seção para oferecer uma visão detalhada das propostas aprovadas pelos vereadores. Também sugeri que a tela inicial funcionasse como um manual de uso, orientando o usuário sobre as funcionalidades do site para facilitar a navegação. Além disso, construí as páginas de "Comentários" e "Estatísticas", com foco em usabilidade e experiência do usuário. Apliquei uma estilização cuidadosa, ajustando o layout para garantir uma interface intuitiva e visualmente atraente, e realizei a revisão do design das demais telas, corrigindo inconsistências e aprimorando a apresentação geral da aplicação.
  • Modelagem Completa do Banco de Dados: Realizei a modelagem do banco de dados em todas as suas etapas – conceitual, lógica e física – e implementei um backup para o repositório.
  • Gestão do Repositório e Controle de Versionamento: Atuei na administração do repositório no GitHub, assegurando um fluxo de trabalho organizado com controle de versões e integração de contribuições de forma coesa. Esta administração garantiu um ambiente de desenvolvimento colaborativo e bem estruturado.
  • Auxílio Técnico na Raspagem de Dados: Apoiei a equipe na raspagem de dados, sugerindo práticas que simplificaram o processo e garantiram a obtenção de informações relevantes e precisas para alimentar as estatísticas da plataforma.
