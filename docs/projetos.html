Projeto API Fatec - Turma DSM 2024 | 02

O projeto tem como objetivo o desenvolvimento de uma plataforma web para o acompanhamento do desempenho dos vereadores em exercício na cidade de São José dos Campos. A ideia é oferecer uma ferramenta acessível e intuitiva para que os cidadãos possam acompanhar a atuação dos vereadores durante o mandato, incluindo presença nas sessões, proposições apresentadas, projetos de lei aprovados, e o posicionamento em votações importantes. A plataforma também permite visualizações gráficas comparando o desempenho entre vereadores e possibilita a interação dos cidadãos com comentários e feedbacks.

Ferramentas utilizadas

  • Python 3+ com o microframework Flask para o desenvolvimento do back-end;
  • MariaDB/MySQL como sistema de gerenciamento de banco de dados;
  • HTML5 para estruturar a informação da aplicação;
  • CSS3 para definir o layout e a renderização da interface;
  • GitHub para controle de versão dos artefatos do projeto;
  • Solução responsiva, garantindo acesso em diferentes dispositivos.

Essas tecnologias foram escolhidas para garantir uma navegação intuitiva, com fácil acesso às informações por meio de buscas e filtros. A plataforma também segue as melhores práticas de usabilidade, promovendo transparência e interatividade entre os cidadãos e os representantes políticos.

Como Product Owner (PO) da API, sou responsável por garantir que os requisitos do produto sejam atendidos, coordenando as entregas com o time de desenvolvimento e assegurando que a solução final atenda às necessidades dos usuários.
